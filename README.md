## 🥗 개선점
* UI/UX 개선
  * UI/UX 디자이너 팀원을 충원해 앱 화면 재구현
* 마이페이지 추가
  * 오늘 섭취한 영양 정보 확인
  * 정보 수정
* 영양 정보 추출 로직 개선
  * OCR API + OPEN API를 통해 추출 정확도 개선   
* CI
* 서버 구축 (✔ 진행 중)
  * firebase를 spring으로 전환 진행 중
